Basic Form Tags

Tag Attribute Description
<form> action Defines the location URL where the the form data should be sent
method Defines the HTTP method use to send the forms data (Generally GET or POST(for more secure data))
<input> id Provides a named identifier that can be hooked by the <label> tag.
type Defines how the input box will render and what type of data it can contain- see Type Attribute Table
value Specifies the value of the <input> element-for text, password, hidden 'types' it provides the initial (default) value of the input field
name Specifies the name of a form element-important for referencing data or particular elements via Javascript
list Used with <datalist> to create a list of selectable options for readers-has autocomplete feature. List name should match <datalist> ID
<label> for Defines a label for the <input> element-providing readable text. Label name must match corresponding 'id' attribute
<textarea> row/column Defines a large textfield, dimensions configured via 'row' and 'column' attributes
<datalist> Used with <input type="text" list=""> and the <option> tag, to create a dropdown list with autocomplete function
<select> Used with syntax <option value=""> to provide values for a dropdown box
<option> Used with <datalist> and <select> to create dropdown boxes

Form Validation Tags

Tag Attribute Description
<input> required Creates an input field that must be completed otherwise form will not submit
min/max Sets a minimum/maximum number of digits for number type field
minlength/maxlength Sets a minimum/maximum number of characters for a text field
pattern Assigns a regular expression (regex) to an input field to strictly control what can be included e.g. [0-9]+ allows only number inputs and [a-zA-Z]{8,} must contain 8 or more upper/lowercase latin characters

Form Input Types

Attribute Description Example Code
text Defines input box which accepts text input
password Defines input box which inputs types characters as asteriks (*)
number Defines input box which accepts numerical input
radio Defines radio buttons to allow selection between two variables
checkbox Defines checkboxes to allow selection off multiple variables
range Defines a ranged slider to choose variables
submit Defines a button by which to send form data
